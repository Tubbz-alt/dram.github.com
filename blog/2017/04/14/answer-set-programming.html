答案集程序设计（Answer set programming）

14 Apr 2017

无意间看到Potassco项目，进而了解到答案集程序设计（Answer set programming）的概念。

总体来说，要学习答案集程序设计，需要有这几方面的基础：逻辑编程、知识表示、稳定模型（Stable model），这样的门槛不低。

以下是一些参考资料：

  • Artficial Intelligence Volume 138：这一卷较为系统地介绍了知识表示和稳定模型，出版于2002年，处于答案集程序设计发展的早期，阅读难度相对适中些；

  • Lparse 1.0 User's Manual ：Smodels的Lparse教程，循序渐进，虽然内容有些旧，部分不适用于Clingo，但依然有很高的参考性，可以作为入门材料；

  • A Tutorial on Hybrid Answer Set Solving with clingo：新近的一篇重实践的Clingo教程，可以作为Clingo快速进阶的起步。
