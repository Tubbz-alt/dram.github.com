Self系统环境接口

08 Dec 2016

  • 获取环境变量

    • os environmentAt:

    • os environmentAt:IfFail:

  • 命令行参数

    • _CommandLine —— 返回以vector组织的命令行参数

  • 执行外部命令

    • os command:

    • os command:IfFail:

    • os command:Executing:IfFail:

  • 读写文件

    • os_file openForReading:

    • os_file openForWriting:

    • f close

  • 日志记录

    • log info:

    • log error:

  • 日期时间

    • time current - 当前时间，其中year为年份，month为月份，date为日期

  • 目录操作

    • os mkdir:Mode - 创建目录，例如os mkdir: 'foo' Mode: 8r755
